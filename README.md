# 模擬login登入
使用PYTHON搭配Selenium 模擬登入github

- 1.先安裝python3的版本
https://www.python.org/downloads/windows/

- 2.下載chromeDriver(與系統對應的版本)
https://sites.google.com/a/chromium.org/chromedriver/home

- 3.寫CODE模擬
